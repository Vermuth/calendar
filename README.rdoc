Реализован календарь событий, функции:

1. Создание, удаление, редактирование и унижтожение событий.

2. Регистрация пользователей.

2. Повторение событий каждый: день, месяц, неделю, год.

3. Создание событий видимых: для всех (людей без регистрации), только для себя
